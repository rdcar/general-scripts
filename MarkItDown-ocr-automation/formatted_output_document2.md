

## Description

This infographic outlines the four essential stages of an Optical Character Recognition (OCR) test process, displayed as a linear progression using color-coded arrows:

1. **Scan the Document** (Blue Arrow): The first step involves capturing the document, likely via a scanner or camera.
2. **Load the Image into the OCR Software** (Green Arrow): Once captured, the image file is imported into the specific software application used for recognition.
3. **Extract the Data** (Orange Arrow): The software processes the image to convert the visual text into digital data.
4. **Verify the Data** (Purple Arrow): The final step involves checking the accuracy of the extracted information to ensure it matches the original source before use.

The background features a subtle gradient and reflections on the floor, giving the graphic a clean, modern presentation style suitable for technical documentation or presentations.
